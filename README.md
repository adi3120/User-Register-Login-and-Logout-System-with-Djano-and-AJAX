# User-Register-Login-and-Logout-System-with-Djano-and-AJAX
***
### To run this project on your localhost .
### follow these steps . 

> 1) **Clone this repo** .
>> 游릭 _git clone https://github.com/adi3120/User-Register-Login-and-Logout-System-with-Djano-and-AJAX.git_ .
> 2) **Go into the directory where the manage.py file is located. Then open terminal there and run this .**
>> 游릭 _python manage.py migrate_ 
> 3) **Create admin account**
>> 游릭 _python manage.py createsuperuser_
> 4) **Make migrations for the app**
>> 游릭 _python manage.py makemigrations_
> 5) **Again migrate those changes**
>> 游릭 _python manage.py migrate_
> 6) **Now run the development server**
>> 游릭 _python manage.py runserver_
